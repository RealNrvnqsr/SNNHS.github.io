ICT (Computer Programming)

The ICT strand focuses on developing students' skills in programming, database management, and software development. Students gain hands-on experience using programming languages like HTML, C++, SQL, and Java.

The program prepares learners for careers in software development, IT support, web development, and computer science-related fields.

An In-depth view of the ICT Curriculum:

Grade 11

C++ - Students learn the fundamentals of programming such as variables, loops, conditionals, and problem-solving logic. C++ helps build strong computational thinking skills and introduces students to how software works behind the scenes.

HTML - Students learn how to structure web pages using HTML. They also discover how HTML works together with CSS for styling and JavaScript for interactivity, forming the foundation of modern web development.

Grade 12

SQL - Students learn how to manage and organize data using databases. SQL teaches how to create, retrieve, and manipulate data, which is essential for real-world applications and information systems.

Java - Students explore object-oriented programming concepts such as classes and objects while building more complex programs. Java strengthens logical thinking and prepares students for advanced software development.

Home Economics (Cookery)

The Home Economics (Cookery) strand develops students' culinary skills through hands-on cooking activities, food preparation, and kitchen management. Students learn proper food safety, nutrition, and professional cooking techniques to prepare them for careers in hospitality and culinary industries.

An In-depth view of the HE Curriculum:

Grade 11

Students are introduced to basic cooking methods, kitchen tools, and food preparation techniques. They learn hygiene, safety practices, and simple recipes that build their foundational culinary skills.

Grade 12

Students progress to advanced cooking techniques, menu planning, and kitchen management. They apply skills learned in Grade 11 to prepare more complex dishes, manage time efficiently, and develop professionalism for culinary careers.
